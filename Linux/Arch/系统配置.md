## 基础配置
### 连接网络
```bash
sudo systemctl enable --now NetworkManager
nmcli device wifi connect SSID --ask
# 输入密码完成连接
```

![[Linux/Arch/attachments/76a1b35c0dfe4972cf410130288482fc_MD5.png]]

### 启用蓝牙服务
```bash
sudo systemctl enable --now bluetooth
```

### 安装声卡驱动
```bash
sudo pacman -S sof-firmware # 安装声卡固件
```

### 安装显卡驱动
```bash
sudo pacman -S  # 安装Intel核显驱动
sudo pacman -S nvidia nvidia-utils nvidia-settings # 安装英伟达显卡独显驱动
```

### 常用软件
```bash
sudo pacman -S fish # fish shell
sudo pacman -S btop # 系统监视工具
sudo pacman -S fastfetch # 获取当前配置信息
sudo pacman -S adobe-source-han-serif-cn-fonts wqy-zenhei # 安装几个开源中文字体。一般装上文泉驿就能解决大多 wine 应用中文方块的问题
sudo pacman -S noto-fonts noto-fonts-cjk noto-fonts-emoji noto-fonts-extra # 安装谷歌开源字体及表情
```

## 安装桌面环境KDE

sudo pacman -S plasma-meta gddm
sudo systemctl enable --now gddm # 设置显示服务自启动并立即启动
```

### 安装常用软件

sudo pacman -S dophine # 资源管理器
```

### 添加32位软件支持
```plain
sudo vim /etc/pacman.conf
```

取消[multilib]一节中两行的注释

### ![[Linux/Arch/attachments/8d4adc2f839f666a7bca8f7f9706b5a8_MD5.png]]
```plain
sudo pacman -Sy
```

### 添加archlinuxcn软件源
```plain
sudo vim /etc/pacman.conf
```

在文件末尾添加

```plain
## CERNET (中国) (ipv4, ipv6, http, https)
## Added: 2023-08-19
## This will redirect you to the nearest China educational mirror site. However, please note that some sites may not support IPv6.
[archlinuxcn]
Server = https://mirrors.cernet.edu.cn/archlinuxcn/$arch
```

导入GPG密钥

```plain
sudo pacman -Sy && sudo pacman -S archlinuxcn-keyring
```

## 通用软件配置
### 安装中文输入法（fcitx5）
```bash
sudo pacman -S fcitx5 # 安装IME输入法框架
sudo pacman -S fcitx5-chinese-addons # 安装拼音输入法插件
sudo pacman -S fcitx5-configtool # 安装输入法UI配置界面工具
sudo pacman -S fcitx5-breeze # 安装输入法主题
```

配置系统环境变量，新建～/.lcoal/environment.d/im.conf文件

```bash
XMODIFIERS=@im=fcitx
```
